# Laravel + Go Role

Web stack for Laravel applications.

## Variables

| Variable | Default | Description |
|----------|---------|-------------|
| `php_version` | `8.3` | PHP version |
| `node_version` | `20` | Node.js version |
| `nginx_web_root` | `/var/www/html` | Web root directory |

## Installs

- Nginx + PHP-FPM
- Composer
- Node.js + Bun
- Go
- Supervisor
