# WINBOX LINUX INSTALL

this install auto path to /opt and give the GUI to menu ubuntu, please download first file on
https://mikrotik.com/download/winbox after that's you can install with this

WINBOX_SRC=~/Downloads/WinBox_Linux => this folder name you put the files

```
WINBOX_SRC=~/Downloads/WinBox_Linux \
curl -fsSL https://raw.githubusercontent.com/naagaraa/devops-ansible/refs/heads/main/script/install-winbox.sh | sh
```
